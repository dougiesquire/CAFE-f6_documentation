CAFE-f6 decadal hindcast/forecast documentation
===============================================

Welcome to the documentation for the CSIRO Climate Analysis Forecast Ensemble decadal hindcast/forecast dataset v6 (CAFE-f6). This documentation contains information about the CAFE-f6 dataset, including:

* details of the dataset
* information about how to access the dataset
* known issues with the data
* preliminary assessment of the prediction skill
* all the code used to generate the contents of this documentation

Please open an issue `here <https://github.com/dougiesquire/Squire_2022_CAFE-f6/issues>`_ if you have any questions after reading this documentation.
