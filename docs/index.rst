Welcome to the documentation for the CSIRO Climate Analysis Forecast Ensemble decadal hindcast/forecast dataset v6 (CAFE-f6). In brief, the CAFE-f6 dataset comprises 96-member decadal climate forecasts initialised at the beginning of every May and Nov over the period 1981-2020.

.. note::
   Please open an issue `here <https://github.com/dougiesquire/Squire_2022_CAFE-f6/issues>`_ if you have any questions after reading this documentation.
